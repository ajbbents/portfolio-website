Meet App

This is a serverless, progressive web application (PWA) with React using a test-driven development (TDD) technique. The application uses the Google Calendar API and Serverless functions (AWS lambda) for the authorization server instead of using a traditional server to fetch upcoming full-stack meetup events.
This is the live demo.

Technologies used:

React, AWS Lambda, serverless, Jest, Cucumber-jest, Puppeteer, Atatus, Recharts

Pokédex Application

A small web application built with HTML, CSS, JavaScript, Jquery and Bootstrap. The app loads Pokémon data from an external API and enables the viewing of data points in detail. The app displays 150 Pokemon characters, and when each Pokemon item is clicked, the following details will be shown: image, height, type, abilities.
This is the live demo.

Technologies used:

HTML, CSS, JavaScript, Bootstrap, Jquery, DOM Interaction, Asynchronous Behavior, API Interaction Modals, Dialogs, Polyfills, fetch, IIFE
